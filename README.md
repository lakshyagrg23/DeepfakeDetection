# DeepFake Detection in Images Using CNN and DCT

## Introduction
This project implements a deepfake detection system for images using a Convolutional Neural Network (CNN) and Discrete Cosine Transform (DCT) to extract image features and detect manipulated media.

## Motivation
Deepfake content has grown substantially, causing ethical and security concerns. This project aims to address the detection of deepfake images efficiently by utilizing frequency domain features through DCT and classifying them with CNN.

## Project Architecture
The system applies DCT to extract features from the image’s frequency domain, which are then fed into a CNN model for classification.
